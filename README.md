## Blood and the Circulatory System in the Human Body

This is a documentation project for the subject ``Blood and the Circulatory System in the Human Body``.

### Getting started
1. **Clone the project**
   - ``git clone https://github.com/phattrienai/blood_circulatory_sys_documentation.git``
   - ``cd blood_circulatory_sys_documentation``
2. **Create a virtual environment and then activate it**
   - python3 -v venv venv
   - source venv/bin/active (for Linux, MACOS)
   - venv\Scripts\activate.bat (for Windows)
3. **Install dependences inside venv**
   - pip install -r requirements.txt
4. **Generate documentation**
   - make html (see sphinx help for more choices)
5. **Open**
   - open file build/html/index.html
